## List Presets

```
GET /presets
```

Returns all available photo variation presets with their full configuration.

### Example

```bash
curl http://localhost:8000/presets
```

## List Templates

```
GET /templates
```

Returns the names of all available document templates.

### Example

```bash
curl http://localhost:8000/templates
```

```json Response
[
  "guia_despacho.html"
]
```
